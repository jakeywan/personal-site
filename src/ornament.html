Ornament by Jan Robert Leegte

Jan's work is beautiful and he has a deep history as an artist, but he's also an excellent programmer. Ornament is a collection of 256 unique Ethereum NFTs with on-chain SVGs. The Ornament is built up of subclusters of bevels and spaces between them, which together can make the ornament very compact or spacious, defining the spread of the composition. We ported a substantial portion of his generative JavaScript code into Solidity so it can run on chain.
